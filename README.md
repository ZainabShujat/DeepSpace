# DeepSpace 🌌

A collaborative study room platform designed around presence, accountability, and focused work.

DeepSpace allows users to create public or private study rooms, invite others, take seats in themed environments, track focus sessions, and collaborate in real time.

## ✨ Features
- Core Features
- Authentication
- Guest Mode
- Public & Private Study Rooms
- Room Codes & Invites
- Shared Focus Sessions
- Real-time Chat
- Activity Feed
- Live Presence Tracking
- Session Timer
- Additional Enhancements
- Library, Café, and Metro themed environments
- Interactive seating system
- Waiting room approval workflow
- Host transfer functionality
- Draggable & resizable timer panel
- User profiles and avatars
- Focus goals/status tracking
- Mobile responsive design

## 🎯 Motivation

A lot of study tools focus on timers and task lists. DeepSpace focuses on presence.

The goal is to recreate the feeling of studying alongside others through shared rooms, visible activity, real-time communication, and accountability.
